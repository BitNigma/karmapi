Deploy a prediction event


  • Any user or community member can initiate prediction topics on the protocol.

  • Deploying the event requires 100 $KART.

  • Detailed reward sharing please refer to the liquidation logic above.

Participate in a prediction event

  1. Mint a prop (mrc20.app)

  2. To join a prediction event, you must deposit $SUI and mint a smart inscription corresponding to the option of the prediction event you wish to participate in, which we refer to as a “prediction tool.”

  3. Participate in a prediction event

  4. Any user can join the prediction topic by holding the corresponding smart inscriptions (prediction tool) to place their vote, then wait for the result.

  5. Result confirmation

  6. After the event liquidation, rewards are distributed according to the results (winners).
